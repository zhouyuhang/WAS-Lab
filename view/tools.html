Almost every browser has provided a very useful developer tools, and they all have similar functionalities. You can view the source code, the database, the network, write commend in cosole, etc. Both developers or hackers can make use of these tools to get more information and malipulate the web applications. We have make use of some of the functions in our previous lab, here we will give a more detailed introduction to some functions.

A more powerful tool is the Integrated Testing Suites, for example Fiddler. They are disigned for many functions, for example, web debugging ,performance testing, HTTP/HTTPS traffic recording, web session manipulation, security testing, etc.
