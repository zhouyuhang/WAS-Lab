  3. Lab1: Use the "Elements" for page editing

When I am developing websites, I have found the browser's developer tools to be very useful and helpful to my work. As Chrome is the most popular browser recently, we will use chrome developer tools for our showcases. For other browsers, they have the similar functionalities.

To open the develop tools, you can go to Menu -> More tools -> Developer tools, or press F12 function key. And you will see something like this:

On the left half of this tab, you see the source html code, including the head and body, usually the javascripts are included in the head or written at the end of body. If you want to see the code of a certain element, simply mouse right click on the element then click "Inspect Element". You can try that with the green button below.

On the right half of this tab, there are css styles for each element, and the computed css for it, and the properties also provides many useful information to you.

For this lab, your are going to use the developer tools to finish two tasks. First try to change the text of the following button to "Hello World" with html on browser, second change the button color to red with css.

To do the first, first right click the green button and click "Inspect element". Then you see the html code of this button, double click on the original text and change it to "Hello World". To do the second task, press the "+" sign near the css styles area on the right half of the tab, then input code background-color: red;

The changes only change the bahavior in your browser, not others. But this function is quite useful as you can see as you edit, so you can save a lot of not having to swap between the web application development environment and the browser.
