What is WAS labs?
WAS labs stands for Web Application Security, this website explore various aspects of web application vulnerabilities and ways to prevent them. The way you learn is innovative, you learn as you take part in the labs. These labs will teach you how to bypass clientside control, attack authentication, attack access control, SQL injection, etc.

How to get start?
You can start from homepage, click "learn more" at one of the security area that you are interested at. Then you will be landed to a page listing all of the labs we provided. Then you can start with the labs by clicking names of the labs.

How to use the labs to learn?
All the labs have the similar pattern on our website, it composed of "Introduction", "The lab content", and "Show Hint". The introduction will give you some background on the lab, and guide on your task. Below the introduction is the lab content, it can be a login form, an input field, etc. Your can interact with the lab content to finish your task. Below the lab content there is always a "Show Hint" button, this button shows you how to solve the problem in the lab. Only use this button when you think you cannot solve the problem.
