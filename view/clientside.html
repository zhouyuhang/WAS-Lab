  2. Bypass Client Side Controls

An application restrict user input in two ways with client-side controls. First it may transfer data from client side to server side using a mechanism that it assumes will prevent the user from modifying the data. Second an application may use measures on the client side to control the user's interaction with the client. However all client-side control cannot be trusted as expected. A better way is always combine client-side controls with server side controls.The following labs will show you the flaws in client side controls.
